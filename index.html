De Reactividad

¿Qué es programación reactiva?

- Paradigma de programación

- Se ocupa de Flujo de Datos

- Genera Emisores

- Facilita la propagación de cambios

¿Qué es programación funcional?

Un poco de código imperativo:

function numExcited(nums) {
  var result = [];
  nums.forEach(function(n) {
    if(n % 2 === 1) {
      result.push(n + '!!!');
    }
  });
  return result;
}

Que es programación Funcional?

La meta es el paralelismo!

- Estados inmutables

- Sin efectos secundarios

- El trabajo se puede escalar facilmente en subprocesos.

(* pun!) 🦊

Recetas para programación Reactiva


import { interval } from 'rxjs';

const observable1 = interval(400);
const observable2 = interval(300);

const subscription =
              observable1.subscribe(x =>
                console.log('first: ' + x)
              );
const childSubscription = observable2.subscribe(x =>
                console.log('second: ' + x)
              );

subscription.add(childSubscription);

setTimeout(() => {
  subscription.unsubscribe();
}, 1000);
