Recetas para programación Reactiva


Definición Observer

- Representa la idea de una colección invocable de valores o eventos futuros.

- El patrón de observador es un patrón de diseño de software en el que un objeto, llamado sujeto.

- Mantiene una lista de sus dependientes, llamados observadores.

- Les notifica automáticamente de cualquier cambio de estado, generalmente llamando a uno de sus métodos.

- Es una colección de devoluciones de llamada que sabe cómo escuchar los valores entregados por el Observable.

👀
Definición Observer 

/* Using a function */
var source = Rx.Observable.create(function (observer) {
    observer.onNext(42);
    observer.onCompleted();

    // Note that this is optional, you do not have to return this if you require no cleanup
    return function () { console.log('disposed'); };
});

var subscription = source.subscribe(
    function (x) { console.log('Next: ' + x); },
    function (err) { console.log('Error: ' + err); },
    function () { console.log('Completed'); });
Definición Observable

- Representa la idea de una colección invocable de valores o eventos futuros.

- El modelo ReactiveX Observable le permite tratar flujos de eventos asincrónicos con el mismo tipo de operaciones simples

👀
Definición Observable 

var source = Rx.Observable.generate(
    0,
    function (x) { return x < 3; },
    function (x) { return x + 1; },
    function (x) { return x; }
);

var subscription = source.subscribe(
    function (x) { console.log('Next: ' + x); },
    function (err) { console.log('Error: ' + err); },
    function () { console.log('Completed'); });
Definición Subscription

- Una suscripción es un objeto que representa un recurso disponible

- Es un observable por lo general

- Posee tambien un metodo un unsubscribe

👀
Definición Subscription 

    var source = Rx.Observable.range(0, 3)

var subscription = source.subscribeOnNext(
  function (x) {
    console.log('Next: %s', x);
  });

// Next: 0
// Next: 1
// Next: 2

Definición Subject

- Un sujeto es una especie de puente o proxy.

- Actúa como observador y como observable

👀
Definición Subject 

var subject = new Rx.Subject();

var subscription = subject.subscribe(
    function (x) {
        console.log('Next: ' + x.toString());
    },
    function (err) {
        console.log('Error: ' + err);
    },
    function () {
        console.log('Completed');
    });

subject.onNext(42);

// => Next: 42

subject.onNext(56);

// => Next: 56

subject.onCompleted();

// => Completed
Definición Scheduler

- Un contexto de ejecución y una estructura de datos para ordenar tareas y programar su ejecución.

- Permiten controlar la estrategia de tiempo de ejecución.

- Articulo profundización de Queue, Asap, Async. Click Aquí

👀
Definición Scheduler 

var disposable = Rx.Scheduler.timeout.scheduleWithState(
     'world',
     function (x) {
          console.log('hello ' + x);
     }
);

// => hello world

var mouseMoved = rx.Observable.fromEvent($document, 'mousemove')
              .map(function (e) {
                var offset = getOffset(textContainer);
                return {
                  offsetX : e.clientX - offset.left,
                  offsetY : e.clientY - offset.top
                };
              })
              .flatMap(function(delta) {
                return rx.Observable.fromArray(_.map($scope.text, function(letter, index) {
                  return {
                    letter: letter,
                    delta: delta,
                    index: index
                  };
                }));
              })
              .flatMap(function(letterConfig) {
                return rx.Observable.timer(letterConfig.index * 100)
                  .map(function() {
                    return {
                      text: letterConfig.letter,
                      top: letterConfig.delta.offsetY,
                      left: letterConfig.delta.offsetX + letterConfig.index * 20 + 15,
                      index: letterConfig.index
                    };
                });
              }).
              subscribe(function(letterConfig) {
                $scope.$apply(function() {
                  $scope.letters[letterConfig.index] = letterConfig;
                });
              });

           $scope.$on('$destroy', function(){
             mouseMoved.dispose();
           });
